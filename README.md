# Jogo da Velha com IA (Python)
Repositório de estudo desenvolvido para explorar a linguagem **Python**, focado na implementação de interface gráfica com **Pygame** e tomada de decisão automatizada utilizando o algoritmo de busca **Minimax**.

## Funcionalidades
- **Interface Gráfica:** Desenvolvida com a biblioteca Pygame para uma experiência interativa.
- **Inteligência Artificial:** Oponente que utiliza o algoritmo Minimax para calcular a melhor jogada.
- **Áudio:** Inclui trilha sonora dinâmica durante a partida.

##  Configuração do Ambiente (VS Code)
Para garantir que o projeto utilize o interpretador Python local e reconheça as bibliotecas instaladas, siga os passos abaixo:
---
1. **Instale a Extensão de Python:**
   - Vá ao ícone de **Extensões** (Ctrl+Shift+X).
   - Busque por "Python" e instale a extensão oficial da Microsoft.

2. **Selecione o Interpretador:**
   - Pressione `Ctrl + Shift + P`.
   - Digite `Python: Select Interpreter` e selecione a opção.
   - Escolha `Enter interpreter path...` -> `Find...`.
   - Navegue até a pasta do projeto e selecione: `JogVelha-Study/python/python.exe`.

## Como Rodar o Jogo

1. Abra o terminal integrado no VS Code (`Ctrl + '` ou `Terminal > New Terminal`).
2. Execute o seguinte comando para iniciar a interface gráfica:
