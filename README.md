Selecione o Interpretador

2. Instale a Extensão de Python
    2.1 Clique no ícone de extensões
    2.2 Digite “Python” na caixa de busca
    2.3 Instale a extensão

3. Digite:
    3.1 “Python: Select Interpreter” e selecione essa opção.

4. Escolha:
    4.1 Enter interpreter path -> Find...

5. Selecione a pasta:
    5.1 JogVelha-Study/python/python.exe
